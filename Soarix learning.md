## üí° Step 1 ‚Äî Roadmap Overview

### 1. Git (Version Control)
üîó [git-scm.com/book](https://git-scm.com/book)  
Git is your foundation for managing and syncing code projects.

**Core steps:**
- Initialize repo (`git init`)
- Commit changes (`git add .`, `git commit -m "message"`)
- Push & pull (`git push`, `git pull`)
- Create & switch branches (`git branch`, `git checkout -b`)

**Mini project:**  
Create a GitHub repo named `learning-journey` to store all your learning code.

---

### 2. Python (Automation & Scripting)
üîó [learnpython.org](https://www.learnpython.org)  
Simple language for automation, backend logic, and cybersecurity tools.

**Steps:**
1. Basics: variables, loops, functions  
2. Intermediate: file handling, modules  
3. Learn libraries like `requests` and `os`  
4. Build small utilities  

**Mini projects:**
- Password generator  
- File organizer  
- Port scanner  

---

### 3. Cybersecurity
üîó [tryhackme.com](https://tryhackme.com)  
Hands-on labs to understand hacking, security testing, and networking.

**Steps:**
- Start with *Intro to Cybersecurity* and *Linux Fundamentals*  
- Move to *Jr Penetration Tester Pathway*  
- Automate small security tasks with Python  

**Mini project:**  
Document TryHackMe labs on GitHub or a personal blog (for your portfolio).

---

### 4. Go (Backend & CLI Development)
üîó [learn-golang.org](https://learn-golang.org)  
A fast, concurrent programming language for backend tools and automation.

**Steps:**
- Learn syntax: variables, structs, maps  
- Use goroutines for concurrency  
- Build simple CLI programs  

**Mini projects:**
- To-Do CLI  
- Port scanner  
- REST API server  

---

### 5. Rust (Systems Programming & Security)
üîó [rust-lang.org/learn](https://www.rust-lang.org/learn)  
Modern, memory-safe, high-performance systems language.

**Steps:**
- Study ‚ÄúThe Rust Book‚Äù  
- Learn ownership, borrowing, lifetimes  
- Practice using Cargo  

**Mini projects:**
- CLI calculator  
- File encryptor  
- Rewrite your Go or Python port scanner in Rust
