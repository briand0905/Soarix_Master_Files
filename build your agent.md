## 4. Example Workflow for Soarix Media

**Scenario:** A client messages you on WhatsApp asking for a new ad.

1. Message enters **n8n**.
2. n8n sends text to **Gemini**.
3. Gemini interprets intention, tone, and requirements.
4. Agent creates:
   - headline  
   - caption  
   - image prompt  
   - recommended posting time  
5. n8n schedules the post or sends the draft for approval.
6. Cloud Run keeps everything running efficiently.

This allows Soarix Media to operate like a **24/7 creative marketing engine**.

## 5. Learning Notes (Beginner-Friendly)

- **n8n**: Think of it like a LEGO set for automations. You drag, connect, and build flows.  
- **Gemini**: The “thinking part” — understands language, writes, creates, plans.  
- **Cloud Run**: The apartment where your agent lives. It expands when many people talk to it and shrinks when idle (saving cost).
